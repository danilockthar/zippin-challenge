# React + TypeScript + Vite

Technical challenge for Zippin.

## Run Locally

Clone the project

```bash
  git clone https://github.com/danilockthar/zippin-challenge
```

Go to the project directory

```bash
  cd zippin-challenge
```

Install dependencies

```bash
  pnpm install
  yarn install
  npm install
```

Run tests

```bash
  pnpm test
  yarn test
  npm run test
```

Start the server

```bash
  pnpm dev
  yarn dev
  npm run dev
```
