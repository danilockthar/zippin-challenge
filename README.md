# React + TypeScript + Vite

Technical challenge for Zippin.

## Run Locally

Clone the project

```bash
  git clone https://github.com/danilockthar/zippin-challenge
```

Go to the project directory

```bash
  cd zippin-challenge
```

Install dependencies

```bash
  pnpm install
```
or
```bash
  yarn install
```


Run tests

```bash
  pnpm test
```
or
```bash
  yarn test
```

Start the server

```bash
  pnpm dev
```
or
```bash
  yarn dev
```
